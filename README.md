# 🏗️ Engineering Consultancy Website

A fullstack web application for an engineering consultancy firm offering services in **Topological Surveying, Geotechnical Investigation, and Structural Design**.

Built using React for the frontend and Django REST Framework for the backend.

---

## 🚀 Features

- Company landing page with services overview  
- Detailed Services section (3 engineering domains)  
- Projects portfolio with filtering  
- About Us page with company details  
- Contact form with enquiry submission  
- Client portal with authentication  
- Admin dashboard for managing data  

---

## 🛠️ Tech Stack

**Frontend:**
- React
- Tailwind CSS
- React Router

**Backend:**
- Django
- Django REST Framework
- JWT Authentication

**Database:**
- SQLite (development)
- PostgreSQL (production)

---

## 📂 Project Structure
frontend/ → React app
backend/ → Django API

### Backend
